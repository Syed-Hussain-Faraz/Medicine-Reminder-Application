# Medicine Reminder Desktop Application

A Python desktop application that allows users to manage medicine schedules and receive automated reminders.

## Features
- User login and signup system
- Add and remove medicines
- View medicine schedule
- Automated reminders
- Desktop notifications

## Technologies Used
- Python
- Tkinter
- APScheduler
- Plyer
- JSON

## Installation

1. Clone the repository

git clone https://github.com/yourusername/medicine-reminder-app.git

2. Install dependencies

pip install -r requirements.txt

3. Test if notifications are working

python test_notifications.py

4. Run the program. You can test it using the dummy data in `data.json`, or delete the file to start with a fresh dataset.

5. Run the program

python medicine_reminder.py
